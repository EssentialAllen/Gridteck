# Gridteck Website

Static multi-page corporate website for Gridteck, a B2B geosynthetics manufacturer.

## Pages

- Home: `index.html`
- About: `about.html`
- Products: `products.html`
- Projects: `projects.html`
- Certifications: `certifications.html`
- Contact: `contact.html`

## Main Assets

- Brand logos: `assets/logos/`
- Images: `assets/images/`
- Fonts: `assets/fonts/`
- Home hero video: `Reference/Factory Video 2.mp4`

## Manual GitHub Upload

Upload the contents of this `Website` folder to the root of a new GitHub repository.

Recommended files and folders to include:

- `index.html`
- `about.html`
- `products.html`
- `projects.html`
- `certifications.html`
- `contact.html`
- `styles.css`
- `script.js`
- `assets/`
- `Reference/Factory Video 2.mp4`
- `.nojekyll`
- `.gitignore`
- `README.md`

The other files inside `Reference/` are design references and are not required for the live website.

## GitHub Pages

After uploading:

1. Open the GitHub repository.
2. Go to Settings > Pages.
3. Set the source to the main branch and root folder.
4. Save and wait for GitHub Pages to publish the site.

No build step is required.
